# 💕 AI Companion

Your personal AI companion — girlfriend, boyfriend, lover, best friend & more.

## Features
- 🤖 Powered by Groq AI (ultra fast)
- 🔄 Round-robin API key rotation (unlimited usage)
- 🧠 Memory system — learns about you over time
- 💾 Chat history saved to GitHub
- 📱 Works on all devices
- 🔒 Private & secure

## Setup
1. Add your Groq API keys to `keys.json`
2. Enable GitHub Pages from repo Settings → Pages → main branch
3. Visit your Pages URL and start chatting!

## Adding More API Keys
Edit `keys.json` and replace any `PLACEHOLDER_GROQ_KEY_X` with a real key.

## Tech Stack
- Pure HTML/CSS/JS (no frameworks)
- Groq API (LLaMA3 model)
- GitHub API for storage
- GitHub Pages for hosting
